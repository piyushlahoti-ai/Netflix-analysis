# 🎬 Netflix Data Analyzer

A Python-based data analysis project that explores the Netflix Movies and TV Shows dataset using **Pandas**, **NumPy**, and **Matplotlib**. The project performs data cleaning, preprocessing, and exploratory data analysis (EDA) to uncover valuable insights about Netflix's content library.

---

## 📌 Features

- 📊 Analyze Movies and TV Shows
- 🧹 Clean and preprocess missing data
- 🌍 Explore content by country
- 🎭 Analyze genre distribution
- ⭐ Compare Movies vs TV Shows
- 📅 Analyze content released over the years
- ⏱️ Study movie durations and TV show seasons
- 📈 Generate visualizations using Matplotlib

---

## 🛠️ Technologies Used

- Python 3
- Pandas
- NumPy
- Matplotlib

## 📂 Dataset

This project uses the **Netflix Movies and TV Shows** dataset.

Dataset columns include:

- Show ID
- Type
- Title
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Listed In (Genre)
- Description

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Netflix-Data-Analyzer.git
```

Move into the project folder:

```bash
cd Netflix-Data-Analyzer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python netflix_analysis.py
```
